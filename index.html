How DSM Works

DSM 2026 is a cohort-based program for practitioners building with AI. Each month we host a live session covering a core topic — from agent deployment to evaluation and multi-agent systems.

Sessions run for 90 minutes and include a presentation, live demonstration, and Q&A. All times shown reflect your selected timezone.

1
Monthly Sessions

One live session per month, typically the second Sunday at 14:00 UTC.

2
Pre-session Reading

Each session has prerequisite materials shared one week in advance.

3
Recordings

All sessions are recorded and made available to cohort members within 48 hours.
